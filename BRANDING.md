# Dabuliu VPN branding

This project is a GPL-3.0 branded derivative of FlClash.

- Product name: **Dabuliu VPN**
- Service portal: `https://vpn.dabuliuai.top:8443/`
- Network core: Mihomo / Clash Meta
- Upstream source: `https://github.com/chen08209/FlClash`

Client credentials are not embedded in the application. Each user imports an
individual subscription or configuration generated by the management panel,
so accounts can be disabled, expired, and metered independently.
